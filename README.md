# The Daily News Lens

## Overview
The Daily News Lens is a dynamic web application that fetches real-time news headlines and weather updates using JavaScript API calls. This project demonstrates my strong understanding of JavaScript functions, API integration, and asynchronous programming.

## Features
- Fetches and displays top news headlines using the **NewsAPI**.
- Fetches and displays real-time weather updates using the **WeatherAPI**.
- Dynamically updates the webpage with news images, titles, descriptions, and source links.
- Implements error handling for API calls.
- Uses modern JavaScript features like `async/await`, `map()`, and `forEach()`.

## Technologies Used
- **HTML** - For structuring the webpage.
- **CSS** - For styling and layout.
- **JavaScript (ES6+)** - For handling API calls, DOM manipulation, and event handling.
- **Axios** - For making HTTP requests to APIs.
- **NewsAPI** - To fetch the latest news headlines.
- **WeatherAPI** - To get real-time weather information.

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/daily-news-lens.git
   ```
2. Navigate to the project directory:
   ```sh
   cd daily-news-lens
   ```
3. Open `index.html` in your browser to view the project.

## How It Works
1. The **NewsAPI** fetches the latest headlines and updates the webpage dynamically.
2. The **WeatherAPI** fetches real-time weather updates based on the given location.
3. JavaScript functions handle API responses, extract relevant data, and update the DOM.
4. Error handling ensures smooth execution even if API calls fail.
